## How to Customize

### Update Email Address
In `index.html`, find the email link and update:
```html
<a href="mailto:yelmajouligmail.com">Send Email</a>
```

### Update Social Links
Modify GitHub and LinkedIn URLs in the social links section.

### Change Colors
In `style.css`, update the CSS variables at the top:
```css
:root {
    --primary-color: #0d47a1;      /* Main color */
    --accent-color: #00bcd4;       /* Accent color */
    /* ... other variables ... */
}
```

### Edit Content
All text content can be edited directly in `index.html`.
